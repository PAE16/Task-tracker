# Task Tracker — Краткое руководство

Кратко:
- Легковесный desktop-трекер задач на PyQt5 с локальной SQLite базой.
- Основные модули:
  - `main.py` — точка входа: инициализация БД, применение темы, запуск окна.
  - `gui.py` — GUI: главное окно, диалоги создания/фильтрации, таблица задач.
  - `database.py` — CRUD для задач и проекты, инициализация/seed тестовых данных.
  - `theme.py` — палитры и глобальный stylesheet; централизует цвета для UI.
  - `charts.py` — окна с Matplotlib для аналитики.

Как запустить:

1. Убедитесь, что установлен Python 3 и зависимости:

```bash
python3 -m pip install -r requirements.txt
```

2. Запустить приложение:

```bash
python3 main.py
```

Где менять тему:
- По умолчанию тема хранится в `QSettings` под ключом `appearance/theme`.
- Через меню "Настройки → Тема" можно переключать `system`/`light`/`dark`.

База данных:
- Файл `tasks.db` создаётся в той же папке при первом запуске.
- Если база пустая, `seed_data()` добавляет демонстрационные проекты и задачи.

Когда менять стили:
- Все цвета и глобальный stylesheet — в `theme.py`. Меняйте цвета в словарях
  `_LIGHT`/`_DARK` и/или правьте `stylesheet_for()`.

Дополнительно:
- Чтобы собрать приложение в один файл используйте `pyinstaller`.
  Пример:

```bash
pyinstaller --onefile --windowed --name "TaskTracker" main.py
```
