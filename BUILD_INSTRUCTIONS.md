## 📦 Task Tracker — Сборка приложения

### ✅ macOS

**Приложение уже собрано!** Находится в папке `dist/`:

```
dist/
├── TaskTracker          ← Исполняемый файл (бинарь)
└── TaskTracker.app      ← macOS приложение (стандартный формат)
```

**Как запустить:**

**Вариант 1: Запустить .app (рекомендуется)**
```bash
open dist/TaskTracker.app
```

**Вариант 2: Запустить бинарь напрямую**
```bash
./dist/TaskTracker
```

**Для распространения:**
- Отправить папку `dist/TaskTracker.app` пользователю
- Пользователь просто откроет её как обычное приложение (двойной клик)

---

### 🪟 Windows

**Для сборки на Windows используй этот скрипт в PowerShell:**

```powershell
# 1. Убедись что Python установлен
python --version

# 2. Установи зависимости
pip install -r requirements.txt

# 3. Собери приложение
python build.py
```

**Что будет создано:**

```
dist/
└── TaskTracker.exe      ← Исполняемый файл Windows
```

**Как запустить:**
- Просто двойной клик по `TaskTracker.exe`
- Или из командной строки: `TaskTracker.exe`

**Для распространения:**
- Отправить файл `dist/TaskTracker.exe` пользователю
- Пользователь просто запустит его (не требует установки Python!)

---

### 📋 Системные требования

**macOS:**
- macOS 10.13+
- Архитектура: Intel (x64) или Apple Silicon (ARM64)

**Windows:**
- Windows 7+
- Архитектура: x86-64

---

### 🛠️ Решение проблем

**Если не запускается на macOS:**
```bash
# Попробуй дать права на исполнение
chmod +x dist/TaskTracker
./dist/TaskTracker
```

**Если появляется ошибка "не могу открыть приложение":**
- На macOS: Система → Безопасность и приватность → Разреши приложение
- Или используй: `open -a TaskTracker` из Terminal

**Если нет базы данных при первом запуске:**
- Приложение автоматически создаст `tasks.db` при первом запуске
- В базе появятся демо-задачи для теста

---

### 📝 Заметки для разработчика

**Пересобрать приложение:**
```bash
# Очистить старые сборки
rm -rf build/ dist/

# Собрать заново
python3 build.py
```

**Если нужна кастомная иконка:**
- Добавить `app_icon.icns` для macOS
- Добавить `app_icon.ico` для Windows
- Обновить `build.py` со строкой: `f"--icon=app_icon",`

**Размер приложения:**
- macOS .app: ~39 MB (включая все библиотеки PyQt5 и Matplotlib)
- Windows .exe: ~50 MB

Это нормально для приложений на PyQt5!
