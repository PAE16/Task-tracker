# 📋 Task Tracker

**Минималистичное приложение для управления задачами** с поддержкой светлой/тёмной темы, аналитикой и кроссплатформной сборкой.

Вдохновлено дизайном компании **Nothing** — чистота, минимализм и функциональность.

---

## ✨ Основные функции

✅ **Управление задачами**
- Создание, редактирование и удаление задач
- Двойной клик для редактирования ячеек
- Отметить как завершено одним кликом
- Поддержка проектов, приоритетов и дедлайнов

✅ **Фильтрация и поиск**
- Фильтр по статусу (To Do, In Progress, Done)
- Фильтр по приоритету (Low, Medium, High, Critical)
- Фильтр по проекту и исполнителю
- Поиск по описанию

✅ **Аналитика**
- 6 типов графиков (Matplotlib)
- Нагрузка на исполнителей
- Просроченные задачи
- Среднее время выполнения
- Распределение статусов
- Временная шкала завершённых задач
- Прогноз выполнения

✅ **Дизайн**
- Nothing-inspired минимализм (чистые линии, оранжевый акцент #ff6900)
- Поддержка светлой и тёмной темы
- Автоматическое определение темы системы (macOS/Windows)
- Тонкие анимации открытия диалогов
- Прозрачные панели

✅ **Удобство**
- Изменяемые размеры колонок (как в Excel)
- Сохранение данных в SQLite
- Демо-задачи при первом запуске
- Запуск на Windows и macOS

## 🚀 Быстрый старт

### Требования
- Python 3.8+
- pip

### Установка

```bash
# Клонировать репозиторий
git clone https://github.com/PAE16/Task-tracker.git
cd Task-tracker

# Создать виртуальное окружение (рекомендуется)
python3 -m venv venv
source venv/bin/activate  # macOS/Linux
# или
venv\Scripts\activate  # Windows

# Установить зависимости
pip install -r requirements.txt
```

### Запуск

```bash
# Прямой запуск
python main.py

# macOS (если есть .app бандл)
./run_mac.sh

# Или напрямую из виртуального окружения
./venv/bin/python main.py
```

## 📦 Сборка для Windows и macOS

### Создание standalone исполняемого файла

```bash
# Сборка (автоматически определяет ОС)
python build.py
```

**Результаты:**
- **macOS**: `dist/TaskTracker.app` и `dist/TaskTracker` (~39 MB)
- **Windows**: `dist/TaskTracker.exe` (~50 MB)

### Распространение пользователям

- **macOS**: Отправить папку `dist/TaskTracker.app` — пользователь откроет двойным кликом
- **Windows**: Отправить файл `dist/TaskTracker.exe` — пользователь запустит двойным кликом

**Не требует установки Python или зависимостей!**

## 💻 Скриншоты функциональности

### Главное окно
- Таблица с задачами (10 колонок)
- Кнопки: Тема, Создать, Фильтр, Аналитика
- Кнопки действий: Редактировать (🟠) и Завершить (✓)

### Диалоги
- **Создание задачи**: Проект, Описание, Исполнитель, Приоритет, Статус, Дедлайн
- **Редактирование**: Все поля редактируемы
- **Фильтрация**: Поиск, Статус, Приоритет, Проект, Исполнитель
- **Аналитика**: 6 интерактивных графиков

### Темы
- **Светлая**: Серый фон (#f5f2eb), белые панели, чёрный текст
- **Тёмная**: Чёрный фон (#050505), серые панели, белый текст
- **System**: Автоматическое определение из ОС

## 📋 Использование

### Создание задачи
1. Нажми кнопку **"Создать"** (оранжевая)
2. Заполни форму (обязателен только описание)
3. Нажми **"Сохранить"**

### Редактирование задачи
**Вариант 1:** Двойной клик на ячейку в таблице
**Вариант 2:** Нажми кнопку **"Редактировать"** (оранжевая) в строке

### Отметить как завершено
Нажми зелёную кнопку **"✓"** в строке задачи

### Фильтрация
1. Нажми **"Фильтр"**
2. Выбери критерии
3. Нажми **"Применить"**

### Просмотр аналитики
1. Нажми **"Аналитика"** (синяя)
2. Выбери тип графика
3. Просмотри статистику

### Смена темы
Нажми кнопку **"Тема"** для переключения между светлой и тёмной темой

## 🔧 Разработка

### Добавление новых полей в задачу

1. Отредактируй схему БД в `database.py`:
```python
CREATE TABLE IF NOT EXISTS tasks (
    ...
    new_field TEXT,  # Добавить здесь
    ...
)
```

2. Обновить форму в `gui.py`
3. Обновить таблицу в `populate_table()`

### Добавление новых графиков

1. Создать класс в `charts.py` (наследовать `ChartBase`)
2. Реализовать метод `draw()`
3. Добавить кнопку в `AnalyticsPopupDialog`

### Смена цветов темы

Отредактируй палитры в `theme.py`:
```python
_LIGHT = {
    "window": "#f5f2eb",
    "accent": "#ff6900",
    # ...
}

_DARK = {
    "window": "#050505",
    "accent": "#ff6900",
    # ...
}
```
