# Task Tracker

Минималистичное desktop-приложение для управления задачами с аналитикой на PyQt5 + SQLite.

Проект ориентирован на простой рабочий процесс: быстро создать задачу, отфильтровать список, отметить выполнение и посмотреть сводные графики.

## Основные возможности

### Управление задачами
- Создание, редактирование и удаление задач.
- Быстрое завершение выбранной задачи через кнопку Выполнить (без открытия редактирования).
- Двойной клик по строке открывает окно редактирования.
- Поддержка проекта, исполнителя, приоритета, статуса и дедлайна.

### Таблица и интерфейс
- Изменяемая ширина колонок (с видимыми разделителями).
- Адаптивная таблица: пропорциональная подстройка колонок при изменении размера окна.
- Отдельно оформленная шапка с кнопками действий.
- Кнопки интерфейса локализованы на русский.

### Фильтрация
- Поиск по описанию/исполнителю.
- Фильтр по статусу, приоритету, проекту и исполнителю.
- Сброс фильтров в один клик.

### Аналитика
- 6 встроенных графиков (Matplotlib):
- Загруженность исполнителей.
- Просроченные задачи по проектам.
- Среднее время выполнения по приоритетам.
- Распределение задач по статусам.
- Закрытые задачи по времени (недели/месяцы).
- Прогноз закрытия задач на 2 периода вперед.
- Улучшенная читаемость подписей: выровненные шрифты, аккуратные легенды, прореживание подписей оси X.
- Для недельных меток используется формат диапазона дат (например, 13.01-19.01).

## Технологии

- Python 3.8+
- PyQt5
- SQLite
- Matplotlib
- NumPy
- PyInstaller

## Быстрый старт

### Установка

```bash
git clone https://github.com/PAE16/Task-tracker.git
cd Task-tracker

python3 -m venv venv
source venv/bin/activate  # macOS/Linux
# или
venv\Scripts\activate    # Windows

pip install -r requirements.txt
```

### Запуск

```bash
python main.py
```

## Сборка

```bash
python build.py
```

Результат:
- macOS: dist/TaskTracker.app и dist/TaskTracker
- Windows: dist/TaskTracker.exe

## Использование

### Создание задачи
1. Нажми Создать.
2. Заполни форму.
3. Нажми Создать в диалоге.

### Редактирование задачи
- Двойной клик по строке.
- Или выбрать строку и нажать Изменить.

### Быстрое завершение
1. Выбери строку в таблице.
2. Нажми Выполнить.

Кнопка Выполнить активна только для незавершенных задач.

### Фильтрация
1. Нажми Фильтр.
2. Укажи параметры.
3. Нажми Применить.

### Аналитика
1. Нажми Аналитика.
2. Выбери нужный график.

## Текущий стиль приложения

- Используется фиксированная светлая тема.
- Цветовой акцент: оранжевый (#ff6900).
- Тема задается в theme.py через LIGHT_THEME.

## Известные вопросы

Разреши запуск в Система -> Безопасность и приватность или запусти бинарник вручную:

```bash
chmod +x dist/TaskTracker
./dist/TaskTracker
```
