## 🎯 Task Tracker — Распространение приложения

### Готовые исполняемые файлы

Приложение собрано и готово к использованию для обеих платформ!

---

## 🍎 macOS

### Запуск приложения

**Способ 1: Двойной клик (самый простой)**
```
dist/TaskTracker.app    ← Просто двойной клик!
```

**Способ 2: Из командной строки**
```bash
./run_mac.sh
```

**Способ 3: Классически**
```bash
open dist/TaskTracker.app
```

### Что находится в dist/

- **TaskTracker.app** — Стандартное macOS приложение (откроется как обычное приложение)
- **TaskTracker** — Исполняемый бинарь (если захочешь запустить напрямую)

### Для раздачи пользователям

1. Скопируй папку `dist/TaskTracker.app`
2. Отправь пользователю
3. Пользователь просто откроет двойным кликом

---

## 🪟 Windows

### Запуск приложения

**Способ 1: Двойной клик**
```
dist/TaskTracker.exe    ← Просто двойной клик!
```

**Способ 2: Из PowerShell/CMD**
```cmd
dist\TaskTracker.exe
```

### Для раздачи пользователям

1. Скопируй файл `dist/TaskTracker.exe`
2. Отправь пользователю
3. Пользователь просто откроет двойным кликом
4. **Не требует установки Python или каких-либо зависимостей!**

---

## 💾 Данные приложения

При первом запуске приложение создаст:
- **tasks.db** — База данных с задачами (в папке приложения)
- Демо-задачи для тестирования

Данные сохраняются автоматически!

## 🐛 Если что-то не работает

**На macOS:**
- "Не могу открыть" → Система → Безопасность → Разреши приложение
- Или используй: `./run_mac.sh`

**На Windows:**
- Если не открывается → Проверь что установлен Visual C++ Redistributable
- Если база не создаётся → Проверь права на запись в папку с приложением

**Общее:**
- При первом запуске могут быть задержки (загружаются библиотеки)
- Иконки тем временно скрыты по дизайну Nothing
- Все функции работают только через кнопки и таблицу
